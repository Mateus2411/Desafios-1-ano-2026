# **2. Mini Loja Virtual**

Utiliza:

- arrays de produtos
- funções
- objeto carrinho
- cálculos de desconto
- controle de estoque
- fluxo completo de compra

Funcionalidades:

- catálogo de produtos
- busca por categoria/nome
- adicionar/remover do carrinho
- sistema de descontos progressivos
- finalizar compra
- relatórios simples
